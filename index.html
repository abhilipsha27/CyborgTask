Building Website in HTML and CSS is much fun ! HTML stands for HyperText Markup Language & CSS means Cascading Style Sheets. While HTML is reposible for displaying the content in the page whereas CSS is the language which we use to style an HTML document. It describes how HTML elements should be displayed.

Know More About HTML and CSS here { Explore ME }

Few common features of HTML are:

  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements.
  • HTML elements tell the browser how to display the content.

The importance of CSS lies in the following points:

  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • External stylesheets are stored in CSS files.
